# Wikipedia Instability Project

This repository contains the Quarto report and supporting outputs for the assignment on Wikipedia instability and data quality.

## Repository structure

- `data/`: input data folder (raw data not included)
- `output/`: figures and tables used in the report
- `docs/`: rendered HTML for GitHub Pages
- `assignment.qmd`: Quarto source file

## Topic

**Instability as a Data Quality Issue in Wikipedia: Evidence from Diversity-Related Topics in English Wikipedia**

## Data availability

The original Wikimedia input data files are not included in this repository because of file size constraints. Place the required files in the `data/` folder before rendering the Quarto document.
